# 업무 기기 보안 규정

## 기기 지급과 사용

업무용 노트북, 휴대전화, 태블릿, 보안 토큰은 회사 업무 수행을 위해 지급되는 자산이다. 임직원은 지급받은 기기를 본인 책임 아래 사용해야 하며, 가족이나 지인에게 빌려주거나 개인 사업 목적으로 사용할 수 없다. 기기 수령 시 자산 번호, 상태, 액세서리 목록을 확인하고 이상이 있으면 IT 헬프데스크에 즉시 알린다.

## 화면 잠금과 인증

업무 기기는 비밀번호, 생체 인증, 다중 인증 등 회사가 요구하는 인증 수단을 사용해야 한다. 자리를 비울 때는 즉시 화면을 잠그고, 자동 잠금 시간은 10분 이하로 설정한다. 비밀번호는 다른 서비스와 재사용하지 않으며, 팀원이나 관리자에게도 공유하지 않는다. 인증 우회 도구나 승인되지 않은 관리자 권한 상승은 금지된다.

## 업데이트와 보안 프로그램

운영체제, 브라우저, 보안 프로그램, 업무용 협업 도구는 최신 보안 업데이트를 적용해야 한다. 회사가 배포한 백신과 장치 관리 프로그램을 삭제하거나 비활성화해서는 안 된다. 업데이트 때문에 업무에 영향이 있으면 IT 헬프데스크에 일정 조정을 요청할 수 있지만, 보안 패치를 장기간 미루는 것은 허용되지 않는다.

## 저장매체와 주변기기

USB 저장매체, 외장하드, 개인 NAS, 개인 프린터는 보안 부서가 승인한 경우에만 업무 자료 저장에 사용할 수 있다. 개인정보와 confidential 문서는 외부 저장매체로 복사할 수 없으며, 예외적으로 필요한 경우 암호화와 반출 기록이 필요하다. 알 수 없는 출처의 USB나 충전 장치를 업무 기기에 연결해서는 안 된다.

## 분실과 도난

업무 기기를 분실하거나 도난당한 경우 임직원은 발견 즉시 팀장, IT 헬프데스크, 보안 부서에 신고해야 한다. 신고에는 마지막 사용 위치, 분실 추정 시각, 저장된 주요 자료, 잠금 상태, 네트워크 접속 여부를 포함한다. IT는 원격 잠금 또는 원격 삭제를 수행할 수 있으며, 임직원은 경찰 신고나 보험 처리에 필요한 자료 제공에 협조한다.

## 반납과 폐기

퇴사, 전보, 장기 휴직, 기기 교체 시 업무 기기는 지정된 기한 안에 IT에 반납한다. 반납 전 개인 파일은 직접 백업하되 회사 자료는 삭제하거나 외부로 반출해서는 안 된다. IT는 반납 기기의 저장장치를 초기화하고 자산 상태를 기록한다. 손상이나 분실이 고의 또는 중대한 과실로 확인되면 비용 부담이 발생할 수 있다.
